# Homebrew Tap for Traycer

Official Homebrew tap for installing Traycer on macOS.

## Install

Install the Traycer CLI:

```bash
brew install traycerai/traycer/traycer
```

Install the Traycer Desktop app:

```bash
brew install --cask traycerai/traycer/traycer-desktop
```

## Update

```bash
brew upgrade traycer
brew upgrade --cask traycer-desktop
```

The Desktop app also supports in-app updates, so the cask is published for
fresh installs and Homebrew-managed upgrade checks.

## About This Tap

The formula and cask in this repository are generated by Traycer's release
pipeline:

- `Formula/traycer.rb` installs the `traycer` CLI.
- `Casks/traycer-desktop.rb` installs the Traycer Desktop app.

Manual edits to generated formula or cask files may be overwritten by the next
release.
